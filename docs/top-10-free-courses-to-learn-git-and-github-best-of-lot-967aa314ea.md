另一方面， **Github 是一个源代码库**，也就是你可以存储你的源代码的地方。

> 在当今世界，程序员的 Git 知识库就像你的 LinkedIn 或脸书账户一样重要。

如果您必须向任何公司提交测试代码，下次请记住这个技巧。
