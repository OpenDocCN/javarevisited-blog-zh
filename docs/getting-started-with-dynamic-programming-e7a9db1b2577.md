# 什么是动态编程？

动态编程主要是对普通递归的优化。每当我们看到一个递归解决方案重复调用相同的输入，我们可以使用[动态编程](/javarevisited/6-best-dynamic-programming-courses-for-coding-interviews-14744060923c)来优化它。其思想是简单地存储子问题的结果，这样我们就不必在以后需要时重新计算它们。这种简单的优化减少了时间…
