对于那些不知道 TypeScript 的人来说，它是微软的一种新的动态类型编程语言，通过添加一些有助于面向对象应用程序开发的新特性，特别是类型的引入，增强了 JavaScript的编程语法。

顾名思义， TypeScript 是类型和 JavaScript 的组合，这意味着现在你的代码中有了合适的变量，比如字符串或整数变量，并且更少需要担心的与类型安全相关的错误。TypeScript的好处是它可以编译成可以在浏览器上运行的本机JavaScript代码，这意味着你现在有了一个更好的语法来编写代码，你也可以利用JavaScript的普遍性。

TypeScript 也在WinJS中使用，WinJS是JavaScript的Windows库。你可以用WinJS为Windows Mobile和Windows 10操作系统制作素材。它还减少了在 JavaScript 代码中很常见的错误，由于JavaScript的动态、非类型化格式，很难调试和排除故障。通过使用TypeScript，你可以在编译时发现许多这样的错误。如果你将一个字符串值赋给一个数值变量，那么 TypeScript编译器会警告你，反之亦然。TypeScript对于大型JavaScript项目特别有用，比如一个有10，000行代码的项目。TypeScript项目比JavaScript代码更容易维护。
