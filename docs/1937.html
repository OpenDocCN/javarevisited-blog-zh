使用Github动作实现持续部署和工作流自动化

Github Actions让开发者可以轻松实现软件工作流程的自动化。

在Github行动之前，我们不得不使用类似于 Jenkins 的CI/CD工具，在每次代码被推送到资源库时触发构建。我们还将这些构建与Sonar之类的代码质量检查插件集成在一起。
