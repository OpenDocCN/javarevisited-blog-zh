## 基本步骤:使用标签

**HTML** 使用**标签**向客户端(浏览器)传达如何显示文本和图像。标签包含在 **< >符号**中。在大多数情况下，你以**开始标签**开始，放入将受该标签影响的一个或多个单词，并在单词串的末尾放置一个**结束标签**。
例如，为一个文档创建一个标题，你可以这样做:
<title>HTML</title>
结束标签通常包含一个**"/**。
HTML 标签不区分大小写。

记住，标题通常**不会出现在文档**本身，而是出现在窗口顶部的一个标题框或栏中。
