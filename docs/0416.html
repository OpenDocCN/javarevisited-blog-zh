创建一个网站涉及三种类型的web开发人员专业化:前端web开发人员、后端web开发人员、全栈web开发人员。

前端web开发

它也被称为客户端开发，涉及到使用 HTML 、 CSS 、 Javascript 、 bootstrap 等等来开发网站的前端。这种前端开发的主要目标是使用户易于交互和使用。

后端web开发

它也被称为服务器端开发。它涉及出于安全目的对外界隐藏的编码实现。它用于为网页提供功能。它还使用数据库，如 MongoDB 、 SQL 、 Postgresql 许多工具用于开发网站的后端，如 Node.js 、 Django 、flask、 python 、 ruby on rails

全栈web开发

它是前端和后端web开发人员的结合。那些能够为客户端和服务器开发软件和网站的人被称为全栈式web开发人员。ffull stack web developer的主要优势是根据需求在前端和后端之间切换。
