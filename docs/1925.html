Kafka大量用于事件流

Apache Kafka是一个事件流平台，数据从一个系统流向另一个系统。其用例包括但不限于， IoT ，实时数据分析，日志聚合与分析，消息代理等。

在本文中，我们将学习如何使用 apache Kafka 作为消息代理。消息代理的角色是将数据从一个系统或服务传递到另一个系统或服务。
