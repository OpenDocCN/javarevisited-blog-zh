使用GitHub操作设置CI/CD

GitHub Action是你想在某个事件中完成的任务。

例如，在Push to a master事件中，您希望构建、运行测试、创建docker映像、push to docker hub并部署到Kubernetes。

构建、运行、创建、推送和部署都是GitHub的动作。
