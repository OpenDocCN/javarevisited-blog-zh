最近有人提醒我，Java 的集合框架中有优先级队列。优先级队列的名字来源于它的第一个用途，即在操作系统中调度工作。它是一个部分有序的列表，这意味着它不必对所有的项目进行排序，而只需确保最没有价值的对象在最前面…
