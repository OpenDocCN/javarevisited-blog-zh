任何深度神经网络都将由三种类型的层组成:

*   输入层
*   隐藏层
*   输出层

**1。输入层**

它接收所有的输入，最后一层是输出层，提供所需的输出。

**2。隐藏图层**

这些层之间的所有层称为隐藏层。可以有 n 个隐藏层。隐藏层和每层中的感知器将取决于您试图解决的用例。

**3。输出层**

它提供所需的输出。
