基本步骤:使用标签

HTML 使用标签向客户端(浏览器)传达如何显示文本和图像。标签包含在 < >符号中。在大多数情况下，你以开始标签开始，放入将受该标签影响的一个或多个单词，并在单词串的末尾放置一个结束标签。
例如，为一个文档创建一个标题，你可以这样做:
<title>HTML</title>
结束标签通常包含一个"/。
HTML标签不区分大小写。
