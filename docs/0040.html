@lombok。数据和MyBatis

原文：https://medium.com/javarevisited/lombok-data-and-mybatis-f43bd8ba0c7?source=collection_archive---------4-----------------------#2019-03-05

使用下面的结果类型，

@Data
public class Some {
    ...
    private List<Sting> keywords;
}

带有以下ResultMap的MyBatis抱怨创建结果。

<collection property="keywords" ofType="string" javaType="list">
  <result column="keyword"/>
</collection>

这是因为@Data注释作为@RequiredArgsConstructor工作，MyBatis不能创建值。

这就是为什么我这样改变了这个类。

//@Data
@ToString
@Setter
@Gettter
