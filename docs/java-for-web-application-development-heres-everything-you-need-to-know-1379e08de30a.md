它是交互式和非交互式页面的混合体。有些网站纯粹是为了陈述事实和信息而开发的。它们可能是非交互式的。而有些页面上传评论和意见，人们可以在上面评论反馈。有些网页是为了销售数字或有形产品而制作的。这些页面也被认为是交互式的。但是考虑到这个时代，页面需要有吸引力，才能产生用户友好的结果。
