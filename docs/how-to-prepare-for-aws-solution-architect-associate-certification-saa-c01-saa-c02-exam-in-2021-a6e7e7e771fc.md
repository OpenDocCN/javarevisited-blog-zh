# 首次通过 AWS 认证的 5 个技巧

现在，您已经熟悉了准备 AWS 解决方案架构师认证的流程，是时候看看一些提示了，它们可以帮助您在第一次尝试时就通过这个著名的考试。这不仅意味着节省你的时间和金钱，也意味着失望。

1.  AWS 认证解决方案架构师考试的及格分数每天都在变化。它可能是 60%或 72%甚至更多，但是你应该总是准备 75%在第一次尝试时通过考试。
