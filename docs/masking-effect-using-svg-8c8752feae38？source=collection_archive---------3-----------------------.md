## 剪辑路径，遮罩

这是一篇关于 SVG 的高级文章。这里我们讨论 SVG 的两个主要概念，即剪辑路径和遮罩。它被用来隐藏图像的一部分，并显示另一部分。有很多这样的使用案例，比如把图像作为文本的背景，应用背景纹理效果，给物体制作图案等等。
