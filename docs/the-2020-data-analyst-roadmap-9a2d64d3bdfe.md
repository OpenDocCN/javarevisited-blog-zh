无论如何，你都无法做到这一点，但数据可视化可以获取原始数据，并将其转换成图表，以便更好地理解你的数据。

# 3.学习统计学

你也可以说，如果你不具备统计技能，你就错过了被这位员工聘用的大好机会。你不能低估学习统计学的力量，因为你要处理大量的数据。

你需要从你的数据中提取更深刻的见解，基于这些数据做出决策，做出预测。
