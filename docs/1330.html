2.课程结构和内容质量

这个庞大的课程包含超过 53个小时的视频内容，目标是没有任何经验的初学者，需要开始学习并应用你将在现实生活中的网站或小企业中学到的东西。那么，让我们来探讨一下本课程的内容:

2.1.前端web开发

创建网站或在线服务时，首先要做的是开发前端部分，以及网站将如何为访问者服务，为此，您可以通过学习 HTML/CSS 开发来开始本课程。

HTML是定义网站结构的语言，而CSS是定制网站的外观，比如颜色和文本大小等等。

2.2.JavaScript简介

创建网站的前端后，无论是使用 HTML/CSS 还是 Bootstrap 你都需要为你的网站添加一些交互性，这也是你在本节将要做的。

JavaScript是一种高级编程语言，它可以让你在网站上添加很多很棒的东西，创造更好的用户体验等等。

2.3.后端web开发

在开发了网站的整个前端之后，你必须创建它的后端，正如我之前所说的，当有人与你的服务进行交互时，这就是幕后的操作。

出于这个原因，您将使用两个javascript框架来实现这个目标，这两个框架分别叫做 Node.js 和 Express.js 。您还将了解API以及如何将其他服务连接到您的网站，如添加支付或通过其他服务发送电子邮件。

2.4.数据库技术

作为一名全栈web开发人员，还需要设计数据库，而不仅仅是创建前端和后端。在这一节中你将如何使用 SQL数据库技术以及其他一些如 MongoDB 和Mongoose来注册用户的数据等等。

在这之后，你将把所有东西放在一起，自己创建整个网站。是不是很棒！

2.5.学习反应

到目前为止，你已经学习了许多编程语言来创建你的网站的界面，像 HTML/CSS 、 Bootstrap 等等，但是实际上还有许多其他的框架来做这件事，其中一个叫做React。

如果你不知道， React 是脸书开发的一个JavaScript框架，用来创建用户界面或者说前端。它是最流行的前端开发框架之一，被世界各地的大公司和小公司所使用。
