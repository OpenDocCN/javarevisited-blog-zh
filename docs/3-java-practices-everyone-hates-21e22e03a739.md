*你使用静态关键字吗？你抛出异常了吗？您是否将数据放在会话中？*

Java 开发人员实践了这些，但不知道要点。他们不知道什么时候使用静态属性。他们不知道如何处理异常。他们把所有的数据都放在会话中。
