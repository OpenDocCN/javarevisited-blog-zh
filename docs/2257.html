介绍

  • 元组是一个数据结构，它被设计用来存储异构对象。
  • 当我们想要存储或返回异构对象时，比如包含姓名、年龄、薪水等的客户对象，这是很有用的。
  • 在 Java 中，我们没有SDK提供的开箱即用的元组数据结构，但是我们可以使用任何一个集合，比如 array 或者 ArrayList 。
