*****这是一本罕见的 Python 书籍，涵盖了 9 个基本的 Python 库，如 Pandas、MatplotLib、Seaborn、Bokeh、Altair、GGPlot、GeoPandas 和 VisPy。总的来说，这是一本非常棒的 Python 书籍，适合初学者和中级 Python 开发人员学习数据可视化。*****
