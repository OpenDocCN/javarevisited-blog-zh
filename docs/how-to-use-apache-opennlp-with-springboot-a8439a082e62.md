Apache OpenNLP 是一个 [Java 库](/javarevisited/20-essential-java-libraries-and-apis-every-programmer-should-learn-5ccd41812fc7)，它使用机器学习来处理自然语言文本。它是在 Apache 许可证版本 2.0 下发布的。

它支持大部分的自然语言处理功能，如分词、句子分割、词性标注、情感分析等等。
