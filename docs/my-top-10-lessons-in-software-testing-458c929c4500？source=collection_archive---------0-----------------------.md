《软件测试的经验教训》是一本书。一本优秀的书！

这本书是由测试大师詹姆斯·巴赫、杰姆·肯纳和布雷特·佩提乔尔德编写的，收集了 293 条经验教训——大大小小的智慧金块。它写于 2002 年，至今仍然非常有用(也许有一些小的例外，比如第 266 课——学习 Perl)。
