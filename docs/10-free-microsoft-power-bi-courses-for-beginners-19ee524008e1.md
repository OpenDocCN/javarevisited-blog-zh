## 11.使用 POWER BI 实现数据可视化【免费】

这是 Udemy 的另一门免费课程，也是上一门课程的讲师，在这门课程中，您可以学习如何使用 Power BI 进行数据可视化。许多人不知道 Power BI 具有一些出色的数据可视化功能，本课程将让您探索这些功能。

Power BI 报告可能只有一页，只有一种视觉效果，也可能有几页都是视觉效果。区分报表*设计者*和报表*消费者*是很重要的。
